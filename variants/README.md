# Infigy variants web widget

## Code

```html
<head>
  <link
    rel="stylesheet"
    href="https://infigy.cz/widgets/variants/style.css"
  />
</head>
<body>
  <iframe
    id="infigy-variants"
    src="https://infigy.cz/variants-widget"
    width="100%"
    height="100%"
    frameborder="0"
  ></iframe>
</body>
```
